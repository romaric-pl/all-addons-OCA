This module adds a code field on leave types.

Additionally, the display name of leave types is calculated as "CODE - Name" if the leave type has a code.

**Table of contents**

.. contents::
   :local:

Usage
=====

#. Navigate to *Time Off > Configuration > Time Off Types*.
#. Select a leave
#. Add a code
