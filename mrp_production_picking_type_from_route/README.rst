This module add an onchange that updates the operation type based on the
product while creating a new MO manually.

**Table of contents**

.. contents::
   :local:

Usage
=====

To use this module, you need to:

#. Go to *Manufacturing > Manufacturing Orders*.
#. Create a new MO.
#. Select a product, the operation type has been updated based on the routes
   configured for that product.
