This module allows you to restrict the "Save as new template" button to
a specific group of users in the email sender wizard form.

**Table of contents**

.. contents::
   :local:

Configuration
=============

To configure this module, you need to:

1. Go to Settings / Users & Companies / Groups
2. Go to the "Save Email As New Template" group form
3. Only the users with this group will have access to the "Save as new
   template" button

Usage
=====

To use this module, you need to:

-  Go to any chatter and click on "Send message", and open the full view
   with the "full composer" button
