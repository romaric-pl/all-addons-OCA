This module allows you to force the manual selection of a Serial Number/lot
in the picking operations. This is useful when you want to ensure that the
right Serial Number/lot is selected for this given operation.

**Table of contents**

.. contents::
   :local:

Configuration
=============

To block a picking based on missing manual Lot selection:

#. In an operation type, check the box
   "Manual Production Lot Selection"
